# Car-eye-Player-android

Car-eye-Player-android 是car-eye 开源团队开发的一个全功能播放器，支持多种音视频格式，支持多种流媒体传输协议，非常高效。

主要功能有：
1）主要流媒体格式播放，如RTMP 或RTSP
2) 支持录像和回放
3）支持静音模式
